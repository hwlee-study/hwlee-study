# HyeWonLee

### ☀️ Intro
3년간 웹 운영 및 개발 후, 현재 웹 프론트 개발을 전문적으로 하기 위해 공부 중입니다.

### 🗂 Project
외부학원 클라이언트 접근 페이지<br>
웹페이지 설정 홈페이지<br>
학원 내 학생 성적 및 관리 페이지

### 🌈 Goal
웹 프론트 개발에 관심이 있으며, ES6와 React 공부 중입니다.
추후 front는 react로 구현하고, Rest API를 통해 서버단 통신을 하는 간단한 프로젝트를 진행할 예정입니다.
